# Pascal-like Language Parser

A parser for a simplified Pascal-like language, built for a Programming Languages course.
Checks whether input source code follows the language's grammar, structured as a recursive
descent parser.

## What's implemented

- Parsing for program header, variable declarations, statement blocks, assignments,
  expressions, and program termination
- Token-based input consumed from a lexer
- A symbol table that tracks declared identifiers
- Semantic checks: detecting undeclared variables and duplicate declarations
- Separate parsing functions per grammar rule (classic recursive descent structure)
